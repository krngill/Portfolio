CSR System

Python-Excel Integration:

Leveraged openpyxl for smooth Python-Excel integration,
providing a powerful tool for managing student data efficiently.
Enabled easy data retrieval and manipulation, showcasing the
dynamic synergy between Python's simplicity and Excel's
organizational capabilities.

User-Friendly Interface:

Implemented an interactive menu system for intuitive user interaction,
offering options such as adding, updating, deleting, viewing, and searching student records.
Prioritized a seamless user experience, making the system accessible to users with varying levels of technological expertise.

Data Integrity Assurance:

Developed robust validation functions:
validate_id_of_Student: Ensures student ID follows alphanumeric patterns.
validate_dates_of_Semester: Verifies semester dates in "MM/DD/YYYY" or "DD-MM-YYYY" formats.
validate_CGPA_of_Student: Validates CGPA format, accepting numeric values with up to two decimal places or "N/A."
Ensured accurate and correctly formatted inputs, maintaining data integrity within the Excel spreadsheet.

Security Layer:

Implemented a password authentication feature:
Users must authenticate with a correct password to access the system,
adding an extra layer of security.
Authentication failure warnings and termination ensure confidentiality
and protect student records from unauthorized access.
